 2 TIPOS DE ESPAÇAMENTO:
 - Margin (exterior, sempre para fora)
 - Padding (interior, sempre para dentro)

 TIPOS DE BLOCOS (BLOCK):
 - div
 - span


 DIV OU QUALQUER ELEMENTO:
 - id (único)
 - class (pode existir várias)

 ID - utilizado para chamar elementos;
 CLASS - utilizada para estilar (CSS) elementos.





 DISPLAY - forma como um elemento se comporta no layout.
 - inline (na mesma linha e só ocupa o espaço necessário na linha, permite váriios lado a lado)//width auto//
 - inline-BLOCK (suporta vários elementos na mesma linha permitindo atribuir largura a cada um deles)//é um layout de várias colunas na mesma linha//
 - BLOCK (ocupa a largura toda disponível e não aceita mais nenhum elemento ao seu lado)//width 100%//
